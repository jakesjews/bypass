# Demo script (weekly forcing function)

## Demo goal (5 minutes)

Show: login → paste note → draft orders → review/edit → export.

## Script

1) “Here’s a synthetic note with meds + labs.”
2) Paste note and run extraction.
3) Point out:
   - drafted meds
   - drafted labs
   - missing-info questions (if any)
4) Edit one field (dose/frequency).
5) Export JSON/FHIR bundle.
6) Close with: “Next week’s one-sentence win condition.”

## One-sentence win condition template

**By next Wednesday, the demo will…**

Example:
- “…correctly draft and group common lab panels and flag fasting requirements.”
